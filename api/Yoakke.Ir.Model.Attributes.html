IAttribute

IAttributeDefinition

Represents an attribute definition that can be put onto different assembly elements.

An attribute in general has arguments and an assigned value, to have a nicer syntax. In general: [attribute_name(arg1, arg2, ...) = assigned_value].

The above form is equivalent to [attribute_name(arg1, arg2, ..., assigned_value)].

IAttributeTarget

An attribute target that can be read from and written to.

IReadOnlyAttributeTarget

An attribute target that can be queried for attributes, but can't attach any.
